가장 큰 복은 주 안에 거해
(E,C)

1- 가장 큰 복은 주 안에 거해 주님과 함께 교통함일세
주 안에 살며 주와 연합해 늘 교통할 때 막힘 없으리

2- 주 안에 살 때 기름 부음과 성령의 인도 얻을 수 있네
기름부음의 풍성 누리고 그 영을 따라 주와 동행해

3- 주 안에 살 때 빛이 비취고 늘 교통할 때 어둠 사라져
빛 순종할 때 생명 자라고 어둠 다 벗고 마음 상쾌해

4- 주 안에 살 때 약함 강해져 늘 교통할 때 사망 이기네
주 안에 살 때 탄식 찬송돼 늘 교통할 때 기쁨 넘치네

5- 주 안에 살 때 사탄 이기고 늘 교통할 때 세상 이기네
주 안에 살 때 죄 권능 잃고 늘 교통할 때 자아 사라져

6- 주 안에 살 때 형상 본받고 늘 교통할 때 생명 충만해
주 안에 살 때 능력을 얻고 늘 교통할 때 풍성 흡수해

7- 주 안에 살길 내 영 사모해 늘 교통하길 성령 강권해
주 안에 흠뻑 적셔짐으로 주님과 조화되기 원하네
